### 微信路由跳转、页面跳转、页面重载

##### [页面跳转文档](https://developers.weixin.qq.com/miniprogram/dev/api/index.html)

```js
wx.navigateTo()     // 重定向
wx.redirectTo()     // 只能打开非 tabBar 页面
wx.switchTab()      // 只能打开 tabBar 页面。
wx.reLaunch()       // 可以打开任意页面。
```
